# Minishell

This project is about creating a simple shell. (Bash)

This shell supports `'`  `"`  `$` `|`  `<`  `>`  ` <<`  `>>`    `ctrl-C` , `ctrl-D` and `ctrl-\` , `builtins` and `history`.

No support for  `\`  `;` `&&` `||` and wildcards.
